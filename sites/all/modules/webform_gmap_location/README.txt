DESCRIPTION
-----------

This module adds «Gmap» component to Webform.
For Drupal 6 download this module: gmap
(https://drupal.org/project/webform_gmap)

Component settings:
  1. values width/height for google map
  2. download marker
  3. Default location marker on the map
  4. Disable/enable autocomplete places field

REQUIREMENTS
------------

Drupal installed on MySQL, SQLite or PostgreSQL database.
Webform 3.x

INSTALLATION
------------

Just download and enable the module.
